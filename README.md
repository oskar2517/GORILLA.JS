# GORILLA.JS

Faithful recreation of the classic QBasic game [Gorillas](https://en.wikipedia.org/wiki/Gorillas_(video_game)) for the browser.

Play it on [gorillas.zone](https://gorillas.zone).


## Features

- Near pixel-perfect recreation of the original game (including all of its quirks).
- WebRTC-based online multiplayer.

## Disclaimer

This is an unofficial fan recreation of QBasic Gorillas. It is not affiliated with, endorsed by, or sponsored by the original creators or rights holders. All rights to the original game remain with their respective owners.
